# Progress Block

Прототип блока Progress для использования в мобильных web-приложениях. Блок предназначен для отображения процесса выполнения операций и их прогресса.

## Описание

Блок Progress представляет собой круговой индикатор прогресса с возможностью управления состоянием через API. Блок спроектирован для легкого переиспользования в других приложениях.

## Технологии

- **JavaScript (ES6+)** - без использования jQuery и других библиотек/фреймворков
- **CSS** - модульная структура стилей
- **HTML5** - без использования шаблонизаторов

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Anastasia-Gorlina/Progress-block.git
cd Progress-block
```

2. Установите зависимости:
```bash
npm install
```

### Запуск

Запустите локальный сервер разработки:

```bash
npm start
```
